## How To Build

```

git clone https://github.com/EpicoinTM/EpicoinTerm.git
cd EpicoinTerm
make
make dependency
make build

```

You can found .dll and .exe files in bin folder.

## Epicoin Terminal

EpicoinTerm is a cli application allowing to use Epicoin.
